# 基于Springboot的理财管理系统 java108

## 项目概述

本项目是基于Spring Boot框架开发的理财管理系统，旨在为用户提供一站式的个人理财服务，同时为管理员提供全面的管理功能。

## 技术栈

- 后端：Spring Boot、MyBatis
- 数据库：MySQL
- 前端：HTML、CSS

## 工具与环境

- 数据库管理工具：Navicat
- 集成开发环境：IDEA

## 系统角色与功能模块

### 用户功能

- 登录与注册
- 系统首页浏览
- 个人理财管理
- 银行推荐服务
- 新闻咨询查看
- 资金记录查询
- 安全网贷投资
- 我的理财管理
- 银行卡管理
- 账户安全管理

### 管理员功能

- 用户管理
- 理财产品管理
- 权限管理
- 网贷管理
- 网贷审核
